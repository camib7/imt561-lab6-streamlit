# NYC 311 Mini Dashboard

## Audience
This dashboard is intended for city staff, operations managers, or analysts who need to quickly understand complaint volume and response performance.

## What users can do
With this app, users are able to:
- Filter complaints by borough, channel, complaint type, and response time.
- View the distribution of response times.
- Compare median response times across boroughs.
- View complaint counts by borough.
- Review the filtered records in a table.
- Download the filtered dataset as a CSV for further analysis.

## Value
This tool helps identify service patterns, highlight delays, and support efficient, data-informed decisions.
